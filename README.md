# ScenarijPro - Web Tehnologije 2025/26
Ovo je projekat rađen na petom semestru u sklopu predmeta Web tehnologije, na Elektrotehničkom fakultetu Univerziteta u Sarajevu (smjer RI)

## Razvojne faze
* **Spirala 1 (Dizajn):** Responzivni UI (HTML/CSS Grid) za projekte, editor i korisnički profil.
* **Spirala 2 (JS):** Analiza uloga i dijaloga 
* **Spirala 3 (API):** Node/Express backend, JSON perzistencija i **Locking** sistem za kolaboraciju.
* **Spirala 4 (Baza):** Migracija na **MySQL (Sequelize)** 

## Tech Stack
* **Server:** Node.js, Express, Sequelize ORM
* **Baza:** MySQL 
* **Klijent:** JavaScript (ES6), HTML5, CSS3
